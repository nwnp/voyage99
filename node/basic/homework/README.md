# 항해99 Bootcamp - Homework

- 게시글 기능 구현(CRUD)
- 댓글 기능 구현(CRUD)
  - 댓글 추가나 수정 API에서 입력란에 입력값이 없을 때, 예외처리까지

## used package

- express
- mongoose
- dotenv
- morgan
- pug
- tailwindcss
- devDependencies: nodemon

## install

```bash
$ npm install
```

## start

```bash
$ npm start
```

### notice

- .env 파일이 없기 때문에 그냥 실행을 하면 에러날 수 있음
